## 概要
Docker・NextJS（React）・Rails7（APIモード）を利用したapi開発

## 技術スタック
【Backend】
- Ruby 3.1.2
- Rails 7.0.4

【Frontend】
- Next.js

【Database】
- MySQL

【Server】
- Nginx

## 機能要件
- ボタンをクリックした際、期待したユーザー情報が保存されること。
